# LocationApp


## Description
Application that lists locations and shows details about the specific location.


## Requirements
**Minimum compatibility**: API level 21

**Target**: API level 28

**Language**: Native to the platform (Kotlin);

**Architecture**: MVVM

**Other Elements:**
- LiveData
- RecycleView
- Coroutines
- Retrofit
